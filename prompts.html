How I Built This

A log of the AI prompts used to generate this entire website, per the assignment requirements.

Prompt 1: "i need to build a website for my class this is the exact assignment: Requirements: 1. You will build a website completely using one of the AI autocoding tools... I was thinking of asking AI to create me a dynamic website..."
Result: Antigravity analyzed the assignment and proposed several project ideas, including a Course Manager. It created an initial implementation plan outlining the steps: build the app, document the prompts, and deploy it publicly.
Prompt 2: "i want to create an interactive website to help me study for my final exam for data structures and algorithms. I can provide a file of our final review"
Result: The AI updated the implementation plan to specifically build a Data Structures and Algorithms study tool. It asked for preferences on the format (flashcards vs quiz) and requested the review file.
Prompt 3: "i want a section for flashcards, multiple choice, and visuals of the structures. I do have a github account."
Result: The AI generated the complete codebase (`index.html`, `style.css`, `script.js`) featuring a modern, dark-mode design with three functional tabs: a 3D flipping Flashcard section, an interactive Multiple Choice quiz with scoring, and a Visuals section utilizing Mermaid.js for beautiful graph rendering. It also generated this specific Prompts page.
Prompt 4: "I want to deploy this as a public website on GitHub Pages."
(Anticipated)
Result: The AI initialized a Git repository, committed all the generated code, and provided the exact terminal commands (or automated the process) to push the repository to GitHub and enable GitHub Pages, resulting in a live, public URL.
