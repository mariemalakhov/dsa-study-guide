    1. Sorting Algorithms

    Insertion Sort

    Inserts elements into a sorted subarray. O(n^2)

    Merge Sort

    Divide & Conquer. Recursively splits and merges. O(n log n)

    graph TD A["[38, 27, 43, 3]"] --> B["[38, 27]"] A --> C["[43, 3]"] B --> D["[38]"] B --> E["[27]"] C --> F["[43]"] C --> G["[3]"]

    Heap Sort

    Uses a Max-Heap structure for O(n log n) sorting.
